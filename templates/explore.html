Feature analysis

On this page, you will find explanation regarding the wine quality dataset. I used the wine quality dataset from UC Irvine's Machine Learning Dataset repositories. The quality ratings were in the range of 1-10. Upon exploratory analysis I found that none of the wine sample had either very high (9-10) quality rating or either very low (1-2). The dataset consists of 11 features which are explained below along with their correlation with the quality. I have plotted graphs to show the average value of each feature w.r.t. each quality point. Some features show a nice trend w.r.t. the quality ratings while other don't. However, for the joint analysis of all the features w.r.t. the quality, we need to use Machine Learning algorithms which can be found on this page.
